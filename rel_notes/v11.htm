Release Notes


The following changes were made for Trash Can version 1.1:

Installation Script:

  • Complete installation is now handled by trash_install.sh alone.
  • Those running under command mode (run level III) can now install if "root" privileges is needed.
  • Now has the ability to update previous Trash Can versions installed under ~/.trash.

Executable:

  • Fixed problem with previous years trash not deleting correctly.
