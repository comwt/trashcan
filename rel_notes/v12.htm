Release Notes


The following changes were made for Trash Can version 1.2:

Installation Script:

  • Added code to allow for ease of future un-installion by USER (gives the trash_uninstall script something to go by).

Executable:

  • Fixed a bug created by the last modification of code (for last year's trash not deleting correctly) where a variable was being reset for the DEL_ARRAY[@] - all works as it should now.

Un-Install Script:

  • This is a new feature of Trash Can
  • You must first re-install Trash Can 1.2 over your previously installed version for it to work
  • Re-Installing will not delete your old trash since the install script recognizes its existance.
