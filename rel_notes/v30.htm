Release Notes


The following changes were made for Trash Can version 3.0:

Installation Script:

Now has the ability to create users on the fly.
Began breaking out duplicate code into "modules".

Un-Install Script:

Now has the ability to administer user trashcan accounts.
Code allows admin "root" to delete a single user account,
selected user accounts, or all user accounts.
Began breaking out duplicate code into "modules".

Other changes:

Both the install and uninstall scripts accept 1 of 2 new options:
   --help
   --debug
